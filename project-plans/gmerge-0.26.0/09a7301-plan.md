# REIMPLEMENT Playbook: 09a7301 — Remove unnecessary `\x7f` key bindings (keybinding chain 1/4)

## Upstream Change Summary

This commit removes unnecessary `\x7f` (DEL character) key bindings that were redundant. The changes:

1. Removes `{ sequence: '\x7f', ctrl: true }` and `{ sequence: '\x7f', command: true }` from `DELETE_WORD_BACKWARD`
2. Removes `{ sequence: '\x7f' }` from `DELETE_CHAR_LEFT`
3. Updates documentation to remove references to these key combinations
4. Updates test files to remove tests for these removed bindings

The `\x7f` character (DEL/127) was being bound alongside the `backspace` key, but this is redundant and causes issues.

**Files changed upstream:**
- `docs/cli/keyboard-shortcuts.md` - Update documentation
- `packages/cli/src/config/keyBindings.ts` - Remove sequence bindings
- `packages/cli/src/ui/keyMatchers.test.ts` - Remove tests for removed bindings
