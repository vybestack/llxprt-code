# Playbook: Add Descriptions to Settings Dialog

**Upstream SHA:** `9172e2831542a08367edd0de431455df12d6199`
**Upstream Subject:** Add description for each settings item in /settings
**Upstream Stats:** 3 files, 242 insertions

## What Upstream Does

Enhances the **Settings Dialog UI** to display **descriptions** for each setting below the setting name. Previously, the dialog only showed the setting label and value; now it shows:

```
● Preview Features (e.g., models)                                        false
  Enable preview features (e.g., preview models).
```

The description is pulled from the `description` field in `settingsSchema.ts` (which already existed but wasn't displayed).

Key changes:
1. **Layout refactor:** Changes from single-row to multi-row layout for each setting (label + description)
2. **Width calculation:** Adds logic to align values consistently by calculating max label/description width
3. **Height calculation:** Updates `maxVisibleItems` calculation from 2 lines per item to 3 lines per item
4. **Test updates:** Adds a new test and updates all snapshots to include descriptions

## LLxprt Adaptation Strategy

LLxprt **has a Settings Dialog** (`packages/cli/src/ui/components/SettingsDialog.tsx` confirmed), so this change applies directly. The upstream logic is straightforward:

1. **Add description field to settings items** (read from schema)

3. **Calculate max width** for consistent value alignment
4. **Update height calculations** to account for 3 lines per item
5. **Update tests and snapshots**

No conceptual differences from upstream — this is a 1:1 UI enhancement.

#### d) Add max width calculation (lines 202-224):
```typescript
// Add this BEFORE generateSettingsItems() or inside the component
const maxLabelOrDescriptionWidth = useMemo(() => {
  const allKeys = getDialogSettingKeys();
  let max = 0;
  for (const key of allKeys) {
    const def = getSettingDefinition(key);
    if (!def) continue;

    const scopeMessage = getScopeMessageForSetting(
      key,
      selectedScope,
      settings,
    );
    const label = def.label || key;
    const labelFull = label + (scopeMessage ? ` ${scopeMessage}` : '');
    const lWidth = getCachedStringWidth(labelFull);
    const dWidth = def.description
      ? getCachedStringWidth(def.description)
      : 0;

    max = Math.max(max, lWidth, dWidth);
  }
  return max;
}, [selectedScope, settings]);
```
