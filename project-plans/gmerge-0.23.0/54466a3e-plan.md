- packages/core/src/hooks/hookRunner.ts — Use name in error messages
- packages/cli/src/ui/components/views/HooksList.tsx — Display name and description
- packages/cli/src/ui/commands/hooksCommand.ts — Use name for display and completion

3. **Update hookPlanner.ts** (packages/core/src/hooks/hookPlanner.ts):

   - Change to:
     ```typescript

       const name = entry.config.name || '';
       const command = entry.config.command || '';
       return `${name}:${command}`;
     }
     ```
   - This ensures hooks with same name+command are deduplicated, but same command with different names are distinct
